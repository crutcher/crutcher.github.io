# WordChipper

[WordChipper][wc_crate] is a Rust library for tokenization and detokenization of text, with a focus on large language models (LLMs). It provides efficient and customizable tokenization algorithms, as well as support for various tokenization schemes used in popular LLMs.

[wc_crate]: https://crates.io/crates/wordchipper
[wc_repo]: https://github.com/crutcher/wordchipper
